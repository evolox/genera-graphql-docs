ProductImage is an entity that represents an image of the product.
Product can have multiple images. Each image can have a type.
Only image with 'TRY_ON' type can be used for try on generation.
There is no strict limitation on the number of images with 'TRY_ON' type.
Please consider that only the first image with 'TRY_ON' type will be used for try on generation.


```graphql
type ProductImage {
  id: ID!
  url: String!
  externalId: String
  parsedBodyData: String
  type: [ProductImageType]
}
```
