Model is one of the main entities in Genera system.
It represents a digital model of the person that can be used for try-on process.


```graphql
type Model {
  id: ID!
  name: String!
  height: Float
  heightUnit: HeightUnit
  weight: Float
  weightUnit: WeightUnit
  skinTone: String
  createdDate: Date!
  images: [ModelImage]
}
```
