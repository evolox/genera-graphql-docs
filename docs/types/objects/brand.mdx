Brand is an entity that represents a brand or organization in Genera system. It can be a Shopify brand or a standalone brand.
All products should be connected to some brand.
The brand pays for each successful generation.


```graphql
type Brand {
  id: ID!
  name: String!
  type: BrandType!
  externalId: String
}
```
